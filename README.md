# API REST de Controle Financeiro

Sistema full stack de controle financeiro desenvolvido com Java Spring Boot no backend e frontend web integrado com dashboard financeiro e gráficos dinâmicos.

---

# 🚀 Tecnologias

## Backend
- Java 21
- Spring Boot 3.5
- Spring Security
- JWT Authentication
- Spring Data JPA
- Hibernate
- Maven

## Frontend
- HTML
- CSS
- JavaScript
- Chart.js

## Banco de Dados
- PostgreSQL
- H2 Database (testes)

## Ferramentas
- Postman
- pgAdmin
- Git
- GitHub
- Render

---

# 📋 Funcionalidades

- Cadastro de usuários
- Autenticação JWT
- Login seguro
- Controle de receitas
- Controle de despesas
- Categorias financeiras
- Dashboard financeiro
- Gráficos financeiros
- Tabela de transações
- API REST completa
- Integração frontend + backend
- Deploy online

# 🖥 Dashboard

O sistema possui:
